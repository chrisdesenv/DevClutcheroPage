Noxus Operational Manual

Core Commands & System Orchestration

# Level 01: Kernel & OS Execution

Dependência 01 // Windows PowerShell rodando como Administrador
> Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
Dependência 02 // Pré-instalação de binários globais (Git, PostgreSQL, Node)
> git --version ; psql --version ; node -v

# Level 02: Repositórios & Sincronia de Versionamento

Dependência 03 // Estar logado no GitHub CLI ou possuir chave SSH configurada
> git status ; git remote -v
Dependência 04 // Ter conexão com a internet e permissão de rede na porta 443/22
> git fetch ; git log origin/main..main
Dependência 05 // Arquivos modificados não-ignorados (trackeados) no diretório local
> git add . ; git commit -m 'Docs updated' ; git push

# Level 03: Data Analytics & SQL Audit

Dependência 06 // Serviço do PostgreSQL rodando localmente
QUERY> \du ; \dt ; \d [TABELA]
Dependência 07 // Estar logado no DB alvo (ex: psql -U postgres -d MEU_BANCO)
QUERY> SELECT [COLUNA], COUNT(*) FROM [TABELA] GROUP BY [COLUNA] HAVING COUNT(*) > 1;

# INFRA: Padrões de Propagação DNS (SLA Big Techs)

// Escopo de propagação global e latência

CLOUDFLARE

GOOGLE CLOUD

# Level 04: Kubernetes & Container Orchestration

Dependência 08 // Cluster Kubernetes Local (K3d / Docker Desktop) Ativo
> kubectl get nodes ; kubectl get namespaces
Dependência 09 // Helm (Gerenciador de Pacotes K8s) Instalado
> helm repo add bitnami https://charts.bitnami.com/bitnami ; helm repo update

# Level 05: Observabilidade & Telemetria Avançada

Dependência 10 // Volumes Persistentes (PV/PVC) configurados no drive D:\
> helm install prometheus prometheus-community/prometheus --namespace monitoring --create-namespace

Prometheus

Coleta e armazena métricas baseadas em séries temporais. Motor central do monitoramento local e cloud.

Loki (Grafana)

Agregação de logs inspirada no Prometheus. Perfeito para capturar os logs dos agentes (HULK, THOR) via Promtail.

Grafana

O painel central. Integra Prometheus, Loki e Zabbix em dashboards visuais dinâmicos.

Zabbix

Monitoramento robusto de infraestrutura clássica, servidores host e redes legadas. Emite triggers via webhook.
