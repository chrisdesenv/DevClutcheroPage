QUERY> SELECT count(*) FROM information_schema.tables WHERE table_schema = 'public';

# INFRA: Padrões de Propagação DNS (SLA Big Techs)

// Escopo de propagação global e latência

CLOUDFLARE

SLA: ~24 HOURS

Rede Anycast permite propagação em minutos, mas a estabilidade total é garantida em 24h.

GOOGLE CLOUD

SLA: ~48 HOURS

Atualizações autoritativas em segundos, mas replicação para ISPs globais requer 2 dias.

AWS ROUTE 53

SLA: ~48 HOURS

Status INSYNC em 60s, porém o cache mundial de ISPs obedece à janela de 48h.

AZURE / MSFT

SLA: 48-72 HOURS

Margem conservadora para garantir que serviços como Office 365 não falhem.
